# State Crash Data Modeling

Project modeling Washington state crash data. Building a predictive model based off this publicly-available data.

Using:

* R version 4.3.2
* tidyverse v2.0.0
* tidymodels v1.1.1
* themis v1.0.2
* doParallel v1.0.17
* discrim v1.0.1
* data.table v1.14.8
* janitor v2.2.0
